Introduction

PHP_Batch, on PHP_Beautifier_Batch::save() and PHP_Beautifier_Batch::get(), call PHP_Beautifier_Batch::getBatchEngine(). This include a file inside /Beautifier/Batch/ with name equal to mode (File or Directory) plus the compression mode, if exists and create a instance of a subclass of PHP_Beautifier_Batch_Engine. For example, for output to a file with .gz compress, PHP_Beautifier_Batch::getBatchEngine() call the file /Beautifier/Batch/FileGz.php and create a instance of PHP_Beautifier_Batch_FilesGz.

The instance of PHP_Beautifier inside Batch should be called with PHP_Beautifier_Batch::callBeautifier(). Some commons methods are included in the definition of PHP_Beautifier_Batch_Engine.
